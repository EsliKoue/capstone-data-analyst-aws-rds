# Présentation du projet Capstone

## Objectif
Ce projet a pour objectif de concevoir une base de données relationnelle sur AWS RDS (PostgreSQL),
d’y stocker des données de vente et de réaliser des analyses SQL afin de produire des indicateurs
exploitables dans Power BI.

## Technologies utilisées
- AWS RDS (PostgreSQL)
- SQL
- DBeaver
- Power BI

## Démarche
1. Création d’une instance PostgreSQL sur AWS RDS
2. Connexion à la base via DBeaver
3. Création des tables clients et ventes
4. Insertion de données simulées
5. Écriture de requêtes SQL analytiques
6. Visualisation des résultats dans Power BI

## Résultats
Les analyses permettent d’identifier les ventes par client, par catégorie de produits
et le chiffre d’affaires généré.
