# Capstone – Data Analyst Cloud (AWS RDS & SQL)

## Description
Ce projet Capstone a été réalisé dans le cadre d’un bootcamp Data & Cloud.
Il consiste à créer une base de données PostgreSQL sur AWS RDS, à analyser
des données de ventes avec SQL et à produire des visualisations dans Power BI.

## Objectifs
- Comprendre le fonctionnement d’une base de données cloud
- Écrire des requêtes SQL analytiques
- Relier les résultats à des visualisations Power BI

## Stack technique
- AWS RDS (PostgreSQL)
- SQL
- DBeaver
- Power BI
- Excel (préparation des données)

## Structure du dépôt
- `sql/` : scripts SQL (création, insertion, analyse)
- `docs/` : documentation du projet
- `powerbi/` : description du tableau de bord
